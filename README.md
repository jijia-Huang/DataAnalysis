# DataAnalysis Platform

互動式資料分析與機器學習訓練平台

## 功能特色

- 📊 **資料分析**：上傳 CSV 檔案並查看資料分布視覺化
- 🎯 **模型訓練**：訓練線性回歸模型（支援單一或多個目標變數）
- 🔮 **模型預測**：使用訓練好的模型進行預測（即將推出）

## 技術棧

- **Python** - 主要程式語言
- **Streamlit** - Web UI 框架
- **Pandas** - 資料處理與分析
- **NumPy** - 數值計算
- **Plotly** - 互動式資料視覺化
- **Scikit-learn** - 機器學習演算法

## 安裝與執行

### 1. 安裝依賴

```bash
pip install -r requirements.txt
```

### 2. 啟動應用程式

**方式一：使用 Streamlit 指令（標準方式）**
```bash
streamlit run app.py
```

**方式二：使用啟動腳本（Windows）**
```bash
run.bat
```
或直接雙擊 `run.bat` 檔案

**方式三：使用 Python 啟動腳本**
```bash
python run.py
```

應用程式會自動在瀏覽器中開啟，預設網址為 `http://localhost:8501`

## 使用說明

### 資料分析頁面

1. 點選左側導航選單中的「📊 資料分析」
2. 點選「選擇 CSV 檔案」按鈕
3. 選擇要分析的 CSV 檔案
4. 系統會自動顯示：
   - 資料預覽（前 10 筆）
   - 統計摘要
   - 資料分布圖表（直方圖）
   - 變數關係圖表（散點圖矩陣）

### 模型訓練頁面

1. 點選左側導航選單中的「🎯 模型訓練」
2. **選擇訓練資料**：
   - 自動使用資料分析頁面的資料，或
   - 上傳新的 CSV 檔案
3. **選擇目標變數**：選擇一個或多個要預測的欄位
4. **選擇訓練演算法**：
   - **線性回歸**：快速穩定，使用最小二乘法
   - **線性回歸梯度下降**：可顯示訓練過程，支援不同損失函式
5. **設定參數**（僅梯度下降）：
   - 損失函式（MSE、MAE、Huber）
   - 學習率、最大迭代次數、收斂容忍度
6. **開始訓練**：點選「開始訓練」按鈕
7. **查看結果**：
   - 訓練損失曲線（梯度下降）
   - 模型參數（係數和截距）
   - 評估指標（R²、MSE、MAE）
8. **保存模型**：輸入模型名稱並保存，供後續預測使用

### 支援的檔案格式

- CSV 檔案（.csv）
- 編碼：UTF-8、BIG5、GB2312、Latin-1
- 分隔符號：逗號（,）、分號（;）、Tab（\t）

### 訓練功能特色

- ✅ 支援單一或多個目標變數（多輸出迴歸）
- ✅ 兩種訓練演算法：線性回歸、梯度下降
- ✅ 可視化訓練過程（損失曲線）
- ✅ 完整的評估指標（R²、MSE、MAE）
- ✅ 模型保存和載入功能
- ✅ 簡單易用的介面，適合非程式人員
