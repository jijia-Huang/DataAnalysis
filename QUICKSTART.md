# 快速開始指南

歡迎使用 DataAnalysis Platform！本指南將幫助您快速上手使用應用程式。

## 啟動應用程式

1. **雙擊** `DataAnalysisPlatform.exe` 檔案
2. **等待** 瀏覽器自動開啟（約 10-30 秒）
3. **開始使用** 應用程式

## 基本使用流程

### 1. 資料分析 📊

**目的**：上傳並分析 CSV 資料檔案

**步驟**：
1. 點擊左側選單的「📊 資料分析」
2. 點擊「選擇 CSV 檔案」按鈕
3. 選擇您的 CSV 檔案
4. 查看資料預覽、統計摘要和視覺化圖表

**功能**：
- ✅ 資料預覽（前 10 筆）
- ✅ 統計摘要（平均值、標準差等）
- ✅ 資料分布圖表（直方圖）
- ✅ 變數關係圖表（散點圖矩陣）

### 2. 模型訓練 🎯

**目的**：訓練機器學習模型來預測目標變數

**步驟**：
1. 點擊左側選單的「🎯 訓練」
2. 選擇訓練資料（可使用資料分析頁面的資料，或上傳新檔案）
3. 設定資料預處理選項（如需要）
4. 選擇目標變數（要預測的欄位）
5. 選擇訓練演算法（線性回歸或梯度下降）
6. 設定訓練參數
7. 點擊「開始訓練」
8. 查看訓練結果並保存模型

**功能**：
- ✅ 支援單一或多個目標變數
- ✅ 自動資料預處理（百分比轉換、千分位轉換）
- ✅ 多種訓練演算法
- ✅ 訓練過程視覺化
- ✅ 模型評估指標（R²、MAE、RMSE 等）

### 3. 模型預測 🔮

**目的**：使用訓練好的模型進行預測

**步驟**：
1. 點擊左側選單的「🔮 預測」
2. 選擇要使用的模型（訓練頁面的模型或已保存的模型）
3. 選擇預測方式：
   - **單筆資料**：手動輸入特徵值
   - **CSV 批次預測**：上傳包含特徵的 CSV 檔案
4. 輸入資料
5. 點擊「執行預測」
6. 查看預測結果並匯出

**功能**：
- ✅ 單筆資料預測
- ✅ CSV 批次預測
- ✅ 預測結果匯出（CSV）
- ✅ 預測結果視覺化

## 範例工作流程

### 範例 1：簡單的資料分析

```
1. 啟動應用程式
2. 進入「📊 資料分析」頁面
3. 上傳 CSV 檔案
4. 查看資料統計和圖表
5. 完成！
```

### 範例 2：完整的訓練和預測流程

```
1. 啟動應用程式
2. 進入「📊 資料分析」頁面
   → 上傳 CSV 檔案
   → 查看資料概況
3. 進入「🎯 訓練」頁面
   → 使用剛才上傳的資料
   → 選擇目標變數
   → 選擇演算法並訓練
   → 保存模型
4. 進入「🔮 預測」頁面
   → 載入剛才訓練的模型
   → 輸入新資料
   → 查看預測結果
5. 完成！
```

## 資料格式要求

### CSV 檔案格式
- **編碼**：UTF-8（建議）或 Big5
- **分隔符號**：逗號（`,`）
- **標題列**：第一列應為欄位名稱
- **資料類型**：
  - 數值欄位：整數或小數
  - 文字欄位：分類變數或字串
  - 日期欄位：YYYY-MM-DD 格式（可選）

### 範例 CSV 檔案

```csv
日期,銷售額,成本,利潤
2025-01-01,10000,6000,4000
2025-01-02,12000,7000,5000
2025-01-03,11000,6500,4500
```

## 提示與技巧

### 💡 資料分析技巧
- 上傳資料前，先檢查 CSV 檔案格式是否正確
- 如果資料有缺失值，系統會顯示警告
- 使用視覺化圖表來了解資料分布

### 💡 訓練技巧
- 選擇合適的目標變數（數值型欄位）
- 如果資料有百分比欄位，啟用「百分比轉換」功能
- 多目標變數訓練可以同時預測多個欄位
- 訓練完成後記得保存模型

### 💡 預測技巧
- 確保預測資料的特徵欄位與訓練資料一致
- 使用 CSV 批次預測可以一次處理多筆資料
- 預測結果可以匯出為 CSV 檔案
